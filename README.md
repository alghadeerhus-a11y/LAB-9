# Random Forest Project

Predicting whether a borrower paid back their loan using Decision Trees and Random Forests.

## Dataset
- `loan_data.csv` — LendingClub loan data from 2007–2010

## Libraries
pandas, numpy, matplotlib, seaborn, scikit-learn

## How to Run
Upload `loan_data.csv` and `02_LAB9.ipynb` to Google Colab and run all cells.
